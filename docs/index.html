Control Loop (50Hz - Every 20ms)

1

Timer ISR Fires

Timer1 interrupt sets a flag every 20ms (50Hz)

2

Read Encoder Snapshot

Atomically read all 4 encoder values with ISR protection

3

Check Target Progress

Compare current position with target encoder ticks

4

Calculate Slowdown Ramp

Reduce speed as target approaches for smooth stopping

5

Update Motor PWM

Send new PWM values to PCA9685 via I2C (400kHz fast mode)

Motor States (TB67H450 H-Bridge)

State IN1 IN2 Effect Current Flow
Forward PWM 0V (LOW) Motor spins forward Yes - proportional to PWM
Backward 0V (LOW) PWM Motor spins backward Yes - proportional to PWM
Brake 5V (HIGH) 5V (HIGH) Short-circuit brake (quick stop) No - motor locked
Coast (IDLE) 0V (LOW) 0V (LOW) Motor freewheels NO CURRENT

Idle State - No Power Consumption

Important: In IDLE state, all motors are in COAST mode with IN1=0, IN2=0. This means NO current flows through the motors and they can spin freely.

Serial Command Reference

Format: COMMAND,param1,param2\n
Baud Rate: 9600
Speed Range: 20-255 (PWM duty cycle)
Ticks: Encoder counts (4320 = 1 wheel revolution)

Movement Commands

FWD,speed,ticks
Move forward for specified encoder ticks
Example: FWD,100,2000
BWD,speed,ticks
Move backward for specified encoder ticks
Example: BWD,100,2000
LEFT,speed,ticks
Strafe left (sideways movement)
Example: LEFT,80,1000
RIGHT,speed,ticks
Strafe right (sideways movement)
Example: RIGHT,80,1000
TURN,speed,ticks
Rotate in place (positive = CCW/left, negative = CW/right)
Example: TURN,60,500

Control Commands

STOP
Emergency stop - applies brake immediately
Example: STOP
READ
Read all 4 encoder values (returns 4 lines)
Response: 1234\n1230\n1228\n1232\n
SYNC
Run calibration routine (~10 seconds, blocking)
Response: DONE (after calibration completes)

Response Messages

Bridge Setup (on imx8mp/Linux SBC)

Install Dependencies

apt-get update
apt-get install -y python3 python3-serial bluez rfcomm
pip3 install pyserial

Copy Bridge Files to Device

# On your PC
scp tools/bt_bridge/*.py user@imx8mp:/opt/bt_arduino_bridge/

Start the Bridge Service

# Manual start
python3 /opt/bt_arduino_bridge/bt_arduino_bridge.py

# Or as systemd service
systemctl enable bt-arduino-bridge
systemctl start bt-arduino-bridge

Connect from PC

# Pair with bridge device
bluetoothctl
> scan on
> pair XX:XX:XX:XX:XX:XX
> trust XX:XX:XX:XX:XX:XX
> quit

# Then use GUI to connect via Bluetooth

Bridge Logs (on imx8mp)

# Service log
cat /tmp/bt_arduino_bridge.log

# Command log (TX/RX with timestamps)
cat /tmp/bt_bridge_commands.log

# Debug log
cat /tmp/bt_bridge_debug.log
Troubleshooting: If Bluetooth is blocked, run: rfkill unblock bluetooth

4
