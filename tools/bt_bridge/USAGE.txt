================================================================================
                    BLUETOOTH-ARDUINO BRIDGE (Python)
================================================================================

PURPOSE
-------
Forward commands from Bluetooth clients to Arduino via USB serial.
Runs on a device with both Bluetooth and USB (Raspberry Pi, Linux PC, etc).


FILES
-----
  bt_arduino_bridge.py   - Main service (Bluetooth setup, connection handling)
  handle_connection.py   - Per-connection handler (serial communication)
  bt_reset.py            - Bluetooth reset utility (for switching PCs)
  install.sh             - System installation script
  uninstall.sh           - System removal script
  test_handler.py        - Test serial communication directly


QUICK START
-----------
1. Connect Arduino via USB

2. Install as service (recommended):
   sudo ./install.sh

3. Or run manually:
   sudo python3 bt_arduino_bridge.py

4. Connect from phone/PC to "Arduino_BT_Bridge"

5. Send commands (e.g., FWD,100,1000)


SERVICE COMMANDS
----------------
  sudo systemctl start bt-arduino-bridge    # Start
  sudo systemctl stop bt-arduino-bridge     # Stop
  sudo systemctl restart bt-arduino-bridge  # Restart
  sudo systemctl status bt-arduino-bridge   # Check status
  sudo journalctl -u bt-arduino-bridge -f   # View logs


LOG FILES
---------
  /tmp/bt_bridge_commands.log   - Commands sent/received (most useful)
  /tmp/bt_arduino_bridge.log    - Service info log
  /tmp/bt_bridge_debug.log      - Detailed debug log

Watch logs:
  tail -f /tmp/bt_bridge_commands.log


LOG FORMAT
----------
  [BT_RX]         Command received from Bluetooth
  [ARDUINO_TX]    Command sent to Arduino
  [ARDUINO_RX]    Response from Arduino
  [BT_TX]         Response sent to Bluetooth client
  [BT_CONNECT]    Client connected
  [BT_DISCONNECT] Client disconnected

Example:
  [2025-12-24 16:30:21.789] [BT_RX] FWD,100,1000
  [2025-12-24 16:30:21.801] [ARDUINO_TX] FWD,100,1000 | bytes=13
  [2025-12-24 16:30:21.910] [ARDUINO_RX] BUSY
  [2025-12-24 16:30:21.920] [BT_TX] BUSY


ARDUINO COMMANDS
----------------
  FWD,<speed>,<ticks>    Move forward
  BWD,<speed>,<ticks>    Move backward
  TURN,<speed>,<ticks>   Turn (positive=left, negative=right)
  STOP                   Emergency stop
  READ                   Read encoder values
  SYNC                   Calibrate motors
  FWDS                   Smart forward with PID

Parameters:
  speed: 20-255
  ticks: encoder ticks (e.g., 1000)


TESTING
-------
1. Test serial directly (without Bluetooth):
   sudo python3 test_handler.py

2. Test with GUI client:
   cd ../
   python3 robot_test.py


SWITCHING PCs / CONNECTION ISSUES
----------------------------------
When switching to a different PC and having connection problems:

1. Run the reset script on the bridge device:
   sudo python3 bt_reset.py

2. On the new PC:
   - Remove "Arduino_BT_Bridge" from paired devices (if exists)
   - Scan for new devices
   - Pair with "Arduino_BT_Bridge"
   - Connect via COM port

Reset options:
  sudo python3 bt_reset.py           # Full reset (unpair all + restart)
  sudo python3 bt_reset.py --soft    # Soft reset (keep pairings, restart)
  sudo python3 bt_reset.py --unpair  # Only unpair all devices
  sudo python3 bt_reset.py --status  # Show current BT status


TROUBLESHOOTING
---------------
Q: Service won't start?
A: Check Bluetooth adapter:
   hciconfig hci0
   sudo rfkill unblock all

Q: Can't find Arduino?
A: Check USB:
   ls -la /dev/ttyACM* /dev/ttyUSB*

Q: Commands not reaching Arduino?
A: Check command log:
   tail -f /tmp/bt_bridge_commands.log

   Should see both [ARDUINO_TX] and [ARDUINO_RX]

Q: PySerial not installed?
A: Install it:
   sudo apt install python3-serial
   # or
   pip3 install pyserial

Q: Permission denied on serial port?
A: Add user to dialout group:
   sudo usermod -a -G dialout $USER

Q: Connection unstable when switching PCs?
A: Run the reset script:
   sudo python3 bt_reset.py
   Then re-pair from the new PC.


DEPENDENCIES
------------

Install all:
